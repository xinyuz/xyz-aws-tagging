# aws-tagging-nodejs

## Purpose
Create tag for all EC2 instances.


## Prerequisite

1. Install Node.js
- https://nodejs.org/en/download/

2. Install aws-sdk
- https://docs.aws.amazon.com/sdk-for-javascript/v2/developer-guide/getting-started-nodejs.html
```
npm install aws-sdk
```


## Usage
Run ```main.js``` by node:
```
node main.js
```
